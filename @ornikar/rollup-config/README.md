# @ornikar/rollup-config

Ornikar rollup config

## Usage

create a file `rollup.config.js`:

```js
'use strict';

const createConfig = require('@ornikar/rollup-config');

module.exports = createConfig();
```

Add `@babel/runtime` in you dependencies.
